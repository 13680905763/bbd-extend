# 快递单号回填插件

bbd后台批量回填快递单号工具，支持 1688、淘宝、微店等平台的物流单号自动抓取与回填。

## 使用说明

请按照以下步骤依次点击插件上的按钮进行操作：

1.  **批量录入单号**
    *   在订单列表页点击此按钮，插件会自动识别当前页面中“有采购编号但无快递单号”的商品行，并准备进行查询。

2.  **打开弹窗**
    *   点击后，插件会根据识别到的商品信息，自动打开对应采购平台（1688/淘宝/微店）的物流详情页面，并自动抓取快递单号。
    *   *注意：请等待页面加载完成及抓取完毕。*

3.  **批量填入**
    *   快递单号抓取完成后，回到 bbd 后台页面点击此按钮，插件会将抓取到的快递单号自动填入对应的输入框中。

4.  **一键勾选**
    *   点击此按钮，自动勾选已填入单号的订单，方便进行后续的批量确认或发货操作。
